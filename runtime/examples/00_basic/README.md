# Basic Examples

These files are meant to read like a compact tutorial for the core language.

Suggested order:

1. `00_introduction.dice`
2. `01_sweeps.dice`
3. `02_sweeps_advanced.dice`
4. `03_functions_and_split.dice`
5. `04_imports_and_reuse.dice`
6. `05_reports.dice`
7. `06_indexing_basics.dice`
8. `07_adaptive_best_choice.dice`

The early files stay close to expressions people would plausibly type by hand. The later files show how those ideas grow into reusable helpers and simple reports.
