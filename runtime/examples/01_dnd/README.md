# D&D Examples

These are substantial user-facing example programs built on `stdlib/dnd`.

They are intentionally not forced into a tutorial order. Treat them as a library of worked examples.

Included themes:

- concrete attack and spell profiles
- AC and slot-level sweeps
- strategy comparisons and winner maps
- recreated online D&D math discussions as exact executable programs

The old `discussions/` split is flattened here so the whole D&D corpus lives in one folder.
