# Examples

This directory is the user-facing example library.

It is organized in three layers:

- `00_basic/`: an ordered mini tutorial for core language features
- `01_dnd/`: larger D&D-focused example programs built on `stdlib/dnd`
- `02_python_extensions/`: an ordered tutorial for extending dice from Python

The `00_basic/` and `02_python_extensions/` folders are intentionally ordered by complexity. Start with the lower-numbered files first.
